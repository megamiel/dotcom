<div class="sentence" id="kaisetsu"><strong>カルノー図</strong>は、行・列それぞれの論理変数の組合せの結果が"真"となる場合に"1"を、"偽"となる場合に"0"を、その該当セルに書きこむことで論理式を図で表す方法です。<br/><br/>カルノー図から論理式を導くには、表の中のすべての"1"が記入されているセルをグループ化して共通項を取り出すのですが、このグループ化は以下のという3つのルールに従って行います。<ol><li>グループ化するすべてのセルの値は"1"であること</li><li>グループ化するセルの数は2<sup>n</sup>であること</li><li>カルノー図の上下の端および左右の端は連続していると考える</li></ol>このルールに従い設問のカルノー図のグループ化を行うと、下図のようにすべての"1"を2つのグループで囲むことができます。<div class="img_margin"><img alt="02a.gif" src="https://www.ap-siken.com/kakomon/04_aki/img/02a.gif"/></div>次にグループごとに共通項を取り出して、その論理積を作ります。<ul><li>赤い枠で囲ったグループは、<span class="ol">A</span> <span class="ol">B</span> <span class="ol">C</span> <span class="ol">D</span>と<span class="ol">A</span> <span class="ol">B</span> C <span class="ol">D</span>なので、共通項は<span class="ol">A</span> <span class="ol">B</span> <span class="ol">D</span>、論理積は<em class="r"><span class="ol">A</span>・<span class="ol">B</span>・<span class="ol">D</span></em>になります。</li><li>青い枠で囲ったグループは、<span class="ol">A</span> B <span class="ol">C</span> D、<span class="ol">A</span> B C D、A B <span class="ol">C</span> D、A B C Dなので、共通項はB D、論理積は<em class="r">B・D</em>になります。</li></ul>最後に"グループごとに生成した論理積"同士の論理和をとることで等価な論理式が完成します。つまり、赤枠グループの論理積である「<span class="ol">A</span>・<span class="ol">B</span>・<span class="ol">D</span>」と青枠グループの論理積である「B・D」の論理和である「<span class="ol">A</span>・<span class="ol">B</span>・<span class="ol">D</span>＋B・D」が適切な式です。</div>
